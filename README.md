# Duolingo Account Creator

Automasi login dan operasi akun Duolingo berbasis Node.js, Express, dan Playwright.

## Menjalankan lokal

1. Install dependency dengan `npm install`
2. Siapkan `config.json` dari template `config.example.json`
3. Jalankan web UI dengan `npm start`

## Menjalankan mode CLI

Gunakan `npm run start:cli`

## File yang sengaja tidak masuk Git

File berikut disimpan lokal dan sudah di-ignore:

- `config.json`
- `accounts.json`
- `sign-in-account.json`
- `vcc.json`
- `.profiles/`
- `debug/`

Lihat panduan di `DEPLOY_RAILWAY.md`.
